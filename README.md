# Valuta CLI Tool

Et simpelt command line værktøj til valutaomregning.

## 1. Klon projektet

git clone <repo-url>
cd MandoPyOne/valuta-cli # Hvis ikke det virker så
cd MandoPyOne
cd valuta-cli

## 2. Opret virtual environment

python -m venv venv

### Aktiver miljøet

Windows:
venv\Scripts\activate

Mac/Linux:
source venv/bin/activate

## 3. Installer dependencies

pip install -r requirements.txt

## 4. Første gang du kører programmet

python main.py --key DIN_API_KEY

## 5. Brug programmet

python main.py

Vælg en af de tilgænglige valutaer

Indtast beløb

Programmet viser omregningen.
